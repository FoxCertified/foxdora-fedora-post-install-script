The extra scripts are optional side scripts you can run if you so desire.

SAFE:
./uninstall-nvidia-drivers.sh
-- Get a new AMD Graphics card and don't want to reinstall the OS? This uninstalls the Nvidia Drivers and reverts Nvidia-specific system changes.

MAY DECREASE STABILITY:

./negativo17-nvidia-graphics-upgrade.sh
-- installs the LATEST Nvidia drivers from the Negativo17 repos. Unknown impact to stability, can't use CUDA or VA-API currently.
