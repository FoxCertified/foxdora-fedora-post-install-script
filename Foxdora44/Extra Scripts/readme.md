The extra scripts are optional side scripts you can run if you so desire.

SAFE:
uninstall-nvidia-drivers
-- Get a new AMD Graphics card and don't want to reinstall the OS? This reverts the install-nvidia-drivers script changes and uninstalls the Nvidia drivers.

MAY DECREASE STABILITY:

terra-mesa-graphics-upgrade.sh
-- installs the LATEST Mesa drivers from the Terra repos. Also installs Terra. Low impact to stability.

negativo17-nvidia-graphics-upgrade.sh
-- installs the LATEST Nvidia drivers from the Negativo17 repos. Unknown impact to stability, can no longer use CUDA.
